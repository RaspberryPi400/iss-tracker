# ISS Tracker
This is a ISS Tracker! It uses an API (Open Notify) to fetch the current longitude and latitude of the ISS (International Space Station).

# Dependencies
ISS Tracker's Python code needs the 'requests' library to run. You can install it by running 'pip install requests' in your terminal.
